Overpass API · Static Mirror

This endpoint serves a pinned snapshot of public OpenStreetMap data for the southern Carpathians (bbox 45.3,22.7,45.8,25.1), refreshed monthly from the main Overpass instance. Read-only.

Example query

Query path: /api/interpreter?data=[out:json];node(around:200,45.5992,24.6134)["amenity"="shelter"];out meta;

Try it

  • /api/interpreter — returns the known hunter-shelter node in Făgăraș and its full changeset history (add 2017, delete 2019).
